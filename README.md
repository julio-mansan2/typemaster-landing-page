### O que aprendi

<br>

- Aplicar mix-blend-mode a certos elementos:

````css

.main .orange-img::before {
    content: '';
    height: 100%;
    width: 100%;
    background: var(--orange-darken);
    position: absolute;
    border-radius: 0.625rem;
    mix-blend-mode: multiply;
}
````
